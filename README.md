Feedstock for [herwig7](https://herwig.hepforge.org/) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Herwig 7: general-purpose Monte Carlo event generator for particle physics

## Architectures

| Architecture | Latest published |
|--------------|------------------|
| linux-amd64 (`linux-64`) | ✅ `7.3.0` |
| linux-arm64 (`linux-aarch64`) | ❌ not published |

_As of the last feedstock render; the badges above are live._


## Install

```bash
conda install -c hep-forge -c conda-forge herwig7
```
